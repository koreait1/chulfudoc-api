## 🔹 구현 기능
- 게시판 API : 게시판 등록, 게시글 등록, 수정, 삭제, 조회
- 파일 API : 파일 업로드, 다운로드, 삭제
- 회원 정보 API : 회원 가입, 로그인, 정보 수정 및 탈퇴
- 병원 정보 API : 응급의료기관 정보 조회

### 🛠️ 기능 설명
- board
게시판 관련 기능을 제공

- file
파일 업로드 및 관리 기능을 제공

- member
회원 등록, 인증, 정보 관리 기능을 제공

- hospital
응급의료기관 정보 조회
